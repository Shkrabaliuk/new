# ADR-0001: Storage format

## Status
Accepted

## Context
Потрібен переносимий формат зберігання текстових матеріалів, сумісний із Git та читабельний без окремої БД.

## Decision
- Основний вміст зберігається у Markdown (`.md`).
- Службові метадані зберігаються у JSON (`.meta.json`) поруч із `.md`.
- Бінарні активи (зображення) зберігаються у каталозі `assets/`.

## Consequences
### Positive
- Простий перегляд історії змін через Git.
- Висока переносимість між платформами.
- Легкий експорт/імпорт.

### Negative
- Потрібна дисципліна для консистентності metadata-файлів.
- Немає вбудованих транзакцій, як у SQL БД.
