# Glossary

- **Document** — один матеріал (файл) з контентом і метаданими.
- **Section** — логічний фрагмент документа (заголовок + блоки під ним).
- **Asset** — файл (зображення, медіа), прив’язаний до документа.
- **Marginalia** — нотатки на полях, отримані зі службового синтаксису (наприклад `((...))`).
- **Block model** — внутрішнє представлення документа як послідовності блоків (paragraph, heading, list, image, note).
- **Pipeline** — детермінована послідовність перетворень тексту до HTML.
- **Semantic stitching** — стратегія підбору “наступного” документа за тематичною близькістю.
- **Recovery log** — журнал, що дозволяє відновити останні зміни при збоях автозбереження.
