# Glossary

- **Marginalia**: текст у форматі `((...))`, що виноситься на поля.
- **Auto-structure**: автоматичне визначення блоків документа з plain text.
- **Typography engine**: модуль нормалізації типографіки.
- **Infinite narrative**: режим читання, де матеріали утворюють безперервний потік.
- **Block model**: внутрішнє представлення документа у вигляді структурних блоків.
